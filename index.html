Rexon Bot

Hai, perkenalkan saya Rexon Saya adalah Robot WhatsApp MultiFungsi yang dibuat oleh Senz. Saya lahir pada tanggal 17 Agustus 1945. Saya di program menggunakan Bahasa Pemrograman JavaScript dan menggunakan runtime NodeJs. Untuk nomer Bot kamu bisa cek pada Icon WhatsApp dibawah, jangan lupa juga support saya dengan mengikuti Instagram pada icon dibawah.
